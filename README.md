Sistem de gestiune a resurselor umane

Cum functioneaza (Tema 1):
3 clase: Angajati, Proiect, Departament
Jocul e simplu, esti HR-ist si ai la dispozitie mai multe optiuni
#1 - Angajezi persoane: Pui un nume, id, salariu, dupa care il adaugi intr-un departament
#2 - Concediezi un anagajat: Alegi departamentul si id-ul, dupa care persoana este concediata
#3 - Maresti/Scazi salariul unui anagajat cu un procent: Alegi departamentul si id-ul, apoi un procent dorit 
#4 - Afiseaza departamentul: ALegi un departament si vezi exact detaliile angajatilor din acel department
#5 - Viabilitate proiect: Creezi un proiect si il asignezi unui departament, daca bugetul alocat proiectului e mai mic decat suma salariilor angajtilor
din departamentul respectiv, atunci acesta nu este realizabil
